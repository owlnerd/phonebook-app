Homepage

This is the homepage of the Phone Book Application.

Welcome

This is a small but fully functional phone book application made as a demo for PDG partners request, displaying the skills and abilities of a candidate for the open junior developer work positions.

Guide

You can reach the phone book app itself by interacting with the PHONEBOOK link in the main bav bar. You can find out more about implementation details and technologies used in the ABOUT section. Authors contact details can be found in the CONTACT section.

Functionality

Phone Book Application allows a user to insert and delete phone book records to and from the database. It can list all the records that are currently in the database, or filter them out by a specified last name.

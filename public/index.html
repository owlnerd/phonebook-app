Homepage

This is the homepage of the Phone Book Application.

Welcome

This is a simple but fully functional phone book application created as a demo for PDG partners, for the purpose of displaying skills and abilities of a candidate for the open junior developer work positions.

Guide

You can reach the phone book app itself by interacting with the PHONEBOOK link in the main navigation bar. You can find out more about implementation details and stack used in the ABOUT section. Contact details can be found in the CONTACT section.

Short description

Phone Book Application allows a user to insert and delete phone book records to and from the database. It can list all the records that are currently in the database, or filter them out by a specified last name.
